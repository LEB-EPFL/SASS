STEADIER-SAILOR (c) Laboratory of Experimental Biophysics EPFL 2017

Notable commands:
   source("C:\\file\\path\\to\\folder.bsh") - execute script from file
   cd(), cat(), dir(), pwd(), etc. - Unix-like shell commands
   frame() - Display a GUI component in a Frame or JFrame.
   javap(Object) - Print the methods and fields of an object, 
       similar to the output of the Java javap command.
   classBrowser() - open the class browser
   
Have fun!
